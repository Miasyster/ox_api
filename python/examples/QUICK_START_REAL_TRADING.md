# 真实交易快速开始

## 第一步：配置账户信息

编辑 `bin/config/config.ini` 文件：

```ini
[user]
acct=你的资金账号
password=你的交易密码
acct_type=0
sh_trade_account=上海股东账号
sz_trade_account=深圳股东账号
```

**账户类型说明：**
- `0` - 现货
- `1` - 期权
- `2` - 期货
- `3` - 信用交易

## 第二步：运行脚本

```bash
cd python
python examples/real_trading.py
```

## 第三步：登录和交易

1. **确认继续**：输入 `yes` 确认使用真实交易系统
2. **选择配置**：选择使用配置文件或手动输入账户信息
3. **登录账户**：等待登录成功
4. **选择操作**：选择下单、撤单或查看订单

## 注意事项

1. ⚠️ **真实交易风险**：所有操作都会产生真实的交易，请谨慎操作
2. **交易时间**：上海/深圳股票交易时间为 9:30-11:30 和 13:00-15:00
3. **最小交易单位**：100 股（一手），数量必须是 100 的整数倍
4. **价格限制**：限价单价格需要符合涨跌幅限制
5. **资金和持仓**：买入需要足够的资金，卖出需要足够的持仓

## 常见错误处理

### DLL 加载失败
- 检查 DLL 文件是否存在
- 确认在 Windows 环境下运行

### 登录失败
- 检查账户信息是否正确
- 确认账户类型选择正确

### 下单失败
- 检查股东账号是否正确
- 确认证券代码正确
- 检查价格和数量是否合法
- 确认账户资金充足
